# FlyTMovilFront

Frontend del proyecto FlyT Movil.

## Descripción
Este repositorio contiene la aplicación frontend de FlyT Movil. Aquí se concentra la interfaz de usuario y la lógica de presentación para la aplicación móvil/web.

## Requisitos
Instala Node.js y un gestor de paquetes (npm o yarn). Si es una app móvil, instala Android Studio / Xcode / Expo según corresponda.

## Instalación
1. Clona el repositorio:
```bash
git clone https://github.com/GermanelGris/FlyTMovilFront.git
cd FlyTMovilFront
```

2. Instala dependencias:
```bash
npm install
# o
yarn install
```

3. Ejecuta en modo desarrollo:
```bash
npm start
# o
yarn start
```
